Invoke the codex-review skill. Runs an adversarial two-model plan review loop: Claude drafts the plan into PLAN.md, OpenAI Codex CLI reviews it as a read-only critic and returns VERDICT:APPROVED or VERDICT:REVISE, Claude revises and re-submits to the same Codex session until approved or MAX_ROUNDS is reached. Human approves the converged plan before any code is written. Use before high-stakes implementation (auth, schema, migrations, payments). Optional args: rounds=N (default 5).
